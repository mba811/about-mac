关于 Mac
====================

>Welcome to 洛桑扎巴’s Book!

>欢迎来到 Mac 的世界。。。

MAC OSX 是苹果公司设计开发的一款广受好评的操作系统。如果尝试总结 MAC 与 Windows 最大的差异，或许关键字在于：注意力。无论如何，Mac 为人们创造了一个新的创意环境，值得你去关注与尝试。
